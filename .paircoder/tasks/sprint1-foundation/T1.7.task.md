## Objective
Configure structlog for JSON logging to file and human-readable console output.

## Acceptance Criteria
- [ ] `datenight/logging.py` has `setup_logging(level, log_file)` and `get_logger(name)`
- [ ] Creates log directory if missing
- [ ] JSON output with level, timestamp (ISO), event fields
- [ ] Log level filtering works
- [ ] 7 tests pass

## Implementation Details

**setup_logging():**
- Create log dir with `Path.mkdir(parents=True, exist_ok=True)`
- Configure structlog processors: contextvars, add_log_level, TimeStamper(iso), StackInfoRenderer, format_exc_info, JSONRenderer
- File handler + stderr handler via stdlib logging

**get_logger():** Returns `structlog.get_logger(name)`

**Tests:**
- test_setup_logging_creates_log_directory (tmp_path)
- test_log_output_is_json
- test_log_contains_level
- test_log_contains_timestamp
- test_log_contains_event
- test_get_logger_returns_bound_logger
- test_log_level_filtering
