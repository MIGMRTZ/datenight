## Objective
Add Bearer token auth middleware to the Worker. All `/api/*` routes require auth; `/health` stays public.

## Acceptance Criteria
- [x] `worker/src/middleware/auth.ts` validates `Authorization: Bearer <token>` against `AUTH_TOKEN` secret
- [x] Returns 401 JSON error for missing/invalid/wrong tokens
- [x] `/health` remains public (no auth required)
- [x] `/api/ping` test endpoint requires valid auth
- [x] 6 tests pass

## Implementation Details

**Middleware:** Hono `createMiddleware` — extract header, check `Bearer ` prefix, compare token to `c.env.AUTH_TOKEN`.

**Route structure in index.ts:**
- Public: `app.get("/health", ...)`
- Protected: `api.use("*", authMiddleware)` on `/api` route group

**Tests:**
- health without auth → 200
- /api/ping without header → 401
- /api/ping with invalid token → 401
- /api/ping with valid token → 200
- malformed header (Basic instead of Bearer) → 401
- case sensitivity (bearer vs Bearer)
