## Objective
Create the Cloudflare Worker project with Hono router, TypeScript, D1/KV bindings (placeholder IDs), and a `GET /health` endpoint.

## Acceptance Criteria
- [ ] `worker/package.json` with hono, wrangler, vitest, typescript deps
- [ ] `worker/tsconfig.json` targeting ES2022/workers
- [ ] `worker/wrangler.toml` with D1 binding (DB), KV binding (CACHE), PLACEHOLDER IDs
- [ ] `worker/src/index.ts` Hono app with `/health` route
- [ ] `worker/src/types.ts` with `Env` interface (DB, CACHE, AUTH_TOKEN)
- [ ] `npm test` passes 4 tests

## Implementation Details

**wrangler.toml:**
- name: "datenight-api"
- compatibility_date: "2024-12-01"
- compatibility_flags: ["nodejs_compat"]
- D1: binding=DB, database_name=datenight, database_id=PLACEHOLDER
- KV: binding=CACHE, id=PLACEHOLDER

**Health endpoint:** Returns `{ status: "ok", timestamp: "<ISO 8601>" }`

**Tests:** health 200, JSON shape, timestamp field, 404 on unknown
